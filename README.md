# Home Décor Website

A modern, elegant home décor e-commerce website built with React.

## Features

- Monochrome color palette for timeless elegance
- Responsive design
- Multiple sections: Home, Collections, Blog, Privacy Policy
- Modern UI with smooth animations

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm build
```

## Tech Stack

- React 18
- React Router DOM
- Lucide React (for icons)
- CSS3 with modern features
